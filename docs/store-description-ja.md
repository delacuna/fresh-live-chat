## 詳細説明文

### Fresh Live Chat — ゲーム配信チャットのネタバレフィルター

ゲームのライブ配信やアーカイブを観ているとき、チャット欄に流れるネタバレや匂わせ、指示コメントに嫌な思いをしたことはないでしょうか。

**Fresh Live Chat** は、YouTubeのゲーム配信チャットをリアルタイムで監視し、ネタバレをはじめとしたこれらのコメントを自動的に非表示にするChrome拡張機能です。


### 主な機能

**2段階AIフィルタリング**
- Stage 1（瞬時）: キーワードマッチによる高速フィルタ
- Stage 2（AI判定）: Claude AIが文脈を理解してネタバレかどうかを精密に判定
- 誤検出を最小限に抑えつつ、見逃しを防ぐ2重チェック構造

**アーカイブ・ライブ両対応**
- アーカイブ（録画配信）のチャットリプレイをフィルタ
- ライブ配信中のリアルタイムフィルタリングにも対応

**ジャンル別テンプレート**
- RPG、推理・ミステリー、アクション・ホラーなどジャンルを選ぶだけで精度が向上
- 攻略ヒント・指示系コメント（「ここちゃんと探索した方がいいよ」「これ負けイベだよ」等）も検出

**ゲーム進行状況に連動**
- ゲームの進行状況（どこまでクリアしたか）を登録すると、すでに通過した部分への言及は非表示にしない
- まだ到達していない部分のネタバレと、すでに通過済みの内容を自動的に区別

**動画タイトルからゲームを自動推測**
- 動画タイトルからプレイ中のゲームを自動的に推測し、より精度の高い判定を実現

**カスタムNGワード**
- 自分だけのNGワードリストを作成し、任意のコメントを即時フィルタ

**3段階のフィルター強度**
- 厳格: ネタバレ＋攻略ヒントをすべてブロック
- 標準: 明確なネタバレのみブロック（デフォルト）
- オフ: フィルタを無効化


### 使い方

1. Chrome Web Storeからインストール（アカウント登録・ログイン不要）
2. YouTubeでゲーム配信のページを開く
3. 右上のFresh Live Chatアイコンをクリックしてポップアップを開く
4. ゲームタイトルまたはジャンルを選択して完了

設定はすぐに反映され、以降は自動でフィルタリングが動作します。


### プライバシーへの配慮

- **データはローカル保存**: 設定・判定結果はすべてお使いのブラウザ内にのみ保存
- **チャット内容はサーバーに保存しない**: AI判定のためにコメントテキストを送信しますが、ログとして保存・記録することは一切ありません
- **APIキー不要**: Anthropic APIキーをユーザーが用意する必要はありません。Fresh Live Chatが安全なプロキシ経由で管理しています
- **アカウント登録不要**: インストールするだけで即使用可能

詳しいプライバシーポリシー: https://github.com/delacuna/fresh-live-chat/blob/main/docs/privacy-policy.md


### ご利用にあたって

AIが文脈を読んでネタバレを判定しますが、すべてのケースで完全にブロックすることは保証できません。対応タイトルとして登録されているゲームではより精度の高い判定が可能です。対応タイトル外のゲームでも、ジャンル別テンプレートやカスタムNGワードを組み合わせることで精度を高めることができます。誤った判定があった場合は、コメント横の「❌ 誤判定」ボタンから報告いただけます（いただいた報告は将来の改善に活用します）。
